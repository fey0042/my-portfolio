Rules:

Click the Start button to start the game, use the up down left and righ button in the keyboard to control the snake direction.

Once the snake eat one apple, it will become longer; if the snake hit the wall or itself, the game is over.

Click the game area to pause the game, click the resume button to resume the game.

Untick the Sound to switch off the music.
